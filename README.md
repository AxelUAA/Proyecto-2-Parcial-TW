# Catálogo Web - Proyecto Parcial 2

Catálogo web con alta de productos, carrito de compras y contacto.

## Tecnologías

- **Frontend:** Angular (SPA)
- **Backend:** Node.js + Express
- **Base de datos:** MySQL

## Secciones

- Inicio
- Catálogo de productos
- Detalle de producto
- Formulario para agregar producto
- Carrito de compras
- Contacto

## Estructura del Proyecto

```
Proyecto-Parcial2/
├── frontend/    # Angular SPA
├── backend/     # API Node.js + Express
└── database/    # Scripts SQL
```

## Instalación

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
ng serve
```
